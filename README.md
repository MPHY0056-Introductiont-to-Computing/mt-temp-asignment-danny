
# Assignment 1: Introduction to GitHub Collaboration

## Objective
In this assignment, you will learn how to collaborate using GitHub by working in teams, making small edits to files, and committing your changes with meaningful commit messages.

---

## Tasks

### Task 1: Form Teams
- Organize yourselves into **teams of 5â€“6 students**.
- Teams must have **no fewer than 5 and no more than 6 members**.
- Assign your team a **team number**.

---

### Task 2: Assign Roles
- Decide who will be the **team leader**.
- The team leader will be responsible for **hosting and managing the repository**.
- All team members must contribute changes so collaboration can be observed.

---

### Task 3: Edit the Guacamole Recipe
- Open the file named **`Guacamole_Recipe.md`**.
- Make **small edits** to the file:
  - Add or remove ingredients
  - Correct spelling
  - Change a word, number, or sentence  
  *(No major changes are required â€” even a small change counts.)*

---

### Task 4: Commit Your Changes
- After editing the file, **commit your changes**.
- Write a **clear and meaningful commit message** that describes what you changed.
  - Example: `Fix spelling in ingredients list`
  - Example: `Add extra lime to recipe`

Each team member should make **at least one commit**.

### Task 6: Add Cake Ingredients
- One team member should find a cake recipe online.

- Commit the change with a meaningful commit message.

### Task 7: Add Images
- Add one or more images to the cake recipe file.
- Make sure the images display correctly in Markdown.
- Commit your changes.
